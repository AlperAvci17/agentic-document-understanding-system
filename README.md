# Agentic Document Understanding System

This project is an Agentic Document Understanding System developed for the SEN4018 course.

The system allows users to upload PDF, TXT, or DOCX documents. It extracts text from uploaded files, preprocesses the text, generates a simple summary, extracts keywords, classifies the document, evaluates the output, and calculates similarity between multiple uploaded documents.

## Features

- Upload PDF, TXT, and DOCX files
- Extract text from uploaded documents
- Clean and preprocess document text
- Generate a basic document summary
- Extract important keywords
- Classify documents into categories
- Make agentic decisions based on document length and number of files
- Evaluate output quality with rule-based scoring
- Calculate similarity between multiple documents
- Display results using a Streamlit web interface

## Technologies Used

- Python
- Streamlit
- pdfplumber
- python-docx
- pandas
- scikit-learn
- TF-IDF
- Cosine Similarity

## How to Run

1. Create and activate a virtual environment.

2. Install required libraries with pip install -r requirements.txt.

3. Run the Streamlit application with streamlit run app.py.
